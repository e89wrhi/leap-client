# Leap Web Client

The Web App is the primary engine for the Leap Client, delivering a fast, responsive, and minimalist interface for the Leap .NET AI SDK documentation, interactive examples, and landing page.

## 🌟 Key Features

- **Interactive Model Switching**: Dynamically test and compare responses between Claude, Gemini, GPT, and Grok.
- **Provider-Agnostic Showcase**: Discover interactive examples demonstrating AI-powered .NET app capabilities.
- **Theme Support**: Seamless transition between light and dark modes using a neutral design system.
- **Modern Aesthetics**: Premium UI with dynamic animations and fluid layouts.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/)
- **UI Components**: Shared `@leap/ui` package using Tailwind CSS v4.
- **Icons**: Lucide React for consistent, scalable iconography.
- **Accessibility**: Semantic HTML and Radix UI primitives for inclusive design.

## 🚀 Getting Started

This app is part of the Leap Client monorepo.

1. Navigate to the web application directory:

   ```bash
   cd apps/web
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```
   Access the app at `http://localhost:3000`.
