# AI Providers

The Leap AI SDK is designed to be fully provider-agnostic. You can switch between different AI services by simply swapping the provider configuration in the pipeline builder.

## Supported Providers

### OpenAI
The default provider supporting the full range of OpenAI models including GPT-4o, GPT-3.5, and the O-series.

```csharp
var leap = LeapClient.Create()
    .UseOpenAi("your-api-key", "gpt-4o")
    .Build();
```

### Anthropic
Native support for Claude 3.5 Sonnet, Opus, and Haiku models.

```csharp
var leap = LeapClient.Create()
    .UseAnthropic("your-api-key", "claude-3-5-sonnet-20240620")
    .Build();
```

### Google Gemini
Native support for Gemini 1.5 Flash and Pro.

```csharp
var leap = LeapClient.Create()
    .UseGoogle("your-api-key", "gemini-1.5-flash")
    .Build();
```

## Custom REST Providers

To implement your own provider, simply create a class that implements `ILeapProvider` from the `Leap.AI.Core.Abstractions` project.

You will need to implement the `GenerateAsync` method:

```csharp
public class MyCustomProvider : ILeapProvider
{
    public async Task<ChatResponse> GenerateAsync(ChatRequest request, CancellationToken cancellationToken = default)
    {
        // Your custom implementation here (e.g., calling a local LLM or another API)
    }
}
```

Then use it in your builder:

```csharp
var leap = LeapClient.Create()
    .UseProvider(new MyCustomProvider())
    .Build();
```
