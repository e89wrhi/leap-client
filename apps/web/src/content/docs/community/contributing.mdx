# Contributing to Leap AI SDK

Thank you for your interest in contributing to the Leap AI SDK! We welcome contributions from the community to help make this the best AI toolkit for .NET.

## How to Contribute

### 1. Identify an Issue or Feature
Check our GitHub Issues to see what's being worked on or to report a bug. If you have a new feature idea, please open a "Feature Request" discussion first.

### 2. Fork and Branch
1. Fork the repository on GitHub.
2. Create a new branch for your feature or fix (e.g., `feature/new-middleware-system`).

### 3. Development Environment
- **IDE**: Visual Studio 2022 or VS Code with the C# Dev Kit.
- **Framework**: .NET 6.0 or .NET 8.0.

### 4. Coding Standards
- Follow standard C# naming conventions.
- Use file-scoped namespaces.
- Ensure all public API members have XML documentation comments.

### 5. Pull Requests
1. Ensure all existing tests pass.
2. Add unit tests for any new functionality.
3. Submit your PR with a clear description of the changes and link any related issues.

## Code of Conduct
Please be respectful and professional in all interactions within our community. We aim to foster a welcoming and inclusive environment for todos.
