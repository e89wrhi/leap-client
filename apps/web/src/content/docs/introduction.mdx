# Leap AI Documentation

Welcome to the official documentation for the **Leap AI SDK**. Leap is a powerful, modern, and type-safe AI SDK for .NET developers, designed to simplify integration with various AI providers while maintaining high performance and flexibility.

## Key Features

- **Fluent Pipeline Builder**: Easily configure your AI client with a human-readable API.
- **Provider Agnostic**: Switch between OpenAI, Anthropic, Google Gemini, and xAI Grok with zero code changes.
- **Structured Data Extraction**: Automatically generate type-safe JSON objects directly from AI responses.
- **Streaming Support**: Real-time token streaming for responsive user interfaces.
- **Middleware Pipeline**: Built-in support for logging, retries, and custom transformations.
