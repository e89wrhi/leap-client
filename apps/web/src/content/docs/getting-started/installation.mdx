# Installation

The Leap AI SDK is available as a NuGet package. You can install it using the .NET CLI or the NuGet Package Manager.

## Prerequisites

- .NET 6.0 or .NET 8.0 SDK
- An API key from a supported provider (OpenAI, Anthropic, or Google)

## Using .NET CLI

Run the following command in your project directory:

```bash
dotnet add package LeapAi.Sdk
```

> [!NOTE]
> As the SDK is currently in development, you might need to append the `--prerelease` flag to get the latest features.
> `dotnet add package LeapAi.Sdk --prerelease`

## Using Package Manager

If you are using Visual Studio, search for `LeapAi.Sdk` in the NuGet Package Manager and check "Include prerelease" if necessary.

## Next Steps

Once installed, proceed to the [Quick Start](/docs/getting-started/quick-start) to send your first message.
